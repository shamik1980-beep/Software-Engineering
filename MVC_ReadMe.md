# Core Java MVC - Student Manager

A simple console-based project demonstrating the MVC (Model–View–Controller) pattern using Core Java only.

## Features
- View all students
- Add student
- Update student
- Delete student

## MVC Structure
- **Model**: `Student`, `StudentRepository`
- **View**: `StudentView` (console UI)
- **Controller**: `StudentController` (handles flow and calls model/view)
- **Entry Point**: `Main`
